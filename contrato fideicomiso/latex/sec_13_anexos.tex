\textit{(Se acompaña calendario anual con distribución serpiente)}

  \item Fechas exactas inicio/fin
  \item Tarifas aplicables
  \item Contacto gestor
\end{itemize}

  \item Uso y almacenamiento
  \item Derechos de acceso y rectificación
  \item Eliminación de datos

  \item Cálculo de cuota anual
