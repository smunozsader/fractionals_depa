# CALENDARIO FRACTIONAL MÉRIDA 2026
## Distribución de 52 Semanas - Sistema Serpiente

### Resumen del Calendario
Este calendario asigna las 52 semanas del año 2026 a los 4 tickets (Ticket A, B, C, D) utilizando el sistema de rotación SERPIENTE (Zig-Zag) que garantiza equidad matemática perfecta.

### Distribución por Propietario:
- **Ticket A**: 13 semanas (7 ALTA + 3 MEDIA + 3 BAJA)
- **Ticket B**: 13 semanas (7 ALTA + 3 MEDIA + 3 BAJA)
- **Ticket C**: 13 semanas (6 ALTA + 4 MEDIA + 3 BAJA)
- **Ticket D**: 13 semanas (6 ALTA + 4 MEDIA + 3 BAJA)

### Leyenda de Temporadas:
- **ALTA** (26 semanas): Diciembre-Marzo, Julio-Agosto 
  - Mayor demanda turística
  - Clima ideal 18-28°C
  - Mejores tarifas: $2,500 MXN/noche
  - Festividades: Navidad, Año Nuevo, Semana Santa, vacaciones escolares

- **MEDIA** (14 semanas): Abril, Mayo parcial, Octubre-Noviembre
  - Demanda moderada
  - Clima agradable
  - Tarifas intermedias: $1,800 MXN/noche
  - Eventos culturales: Carnaval, festivales locales

- **BAJA** (12 semanas): Mayo-Junio, Septiembre
  - Temporada de lluvias
  - Menor demanda
  - Tarifas bajas: $1,200 MXN/noche
  - Menores multitudes, paisajes exuberantes

### Cómo Funciona el Sistema Serpiente:

**Ronda 1 (Semanas 1-4 de cada temporada)**: A → B → C → D (orden normal)
**Ronda 2 (Semanas 5-8 de cada temporada)**: D → C → B → A (orden invertido) ← ¡D elige PRIMERO ahora!
**Ronda 3 (Semanas 9-12 de cada temporada)**: A → B → C → D (orden normal)
**Ronda 4 (Semanas 13-16 de cada temporada)**: D → C → B → A (orden invertido)

Esta alternancia garantiza que:
- Nadie SIEMPRE elige primero
- Nadie SIEMPRE elige último
- Todos tienen 13 semanas distribuidas equitativamente
- La diferencia de 1 semana en temporada ALTA se compensa con 1 semana MEDIA adicional

### Notas Operativas:
1. El calendario es FIJO para todo el año - no se modifica unilateralmente
2. Intercambios entre propietarios permitidos con acuerdo mutuo documentado
3. Notificación de uso: 60 días de anticipación (deadline 1 de noviembre)
4. Semanas no reclamadas se rentan automáticamente
5. Pago de rendimientos: enero del año siguiente
6. En años bisiestos (53 semanas), semana 53 va al fondo común de rentas
